소개

인프라를 직접 손으로 만들어보고 부수면서 배우는 걸 좋아합니다.
AWS 상에서 3-Tier 아키텍처, VPC/NAT/OpenVPN, ECS + ALB + ECR, GitLab & GitHub를 이용한 Terraform 기반 자동 배포 등을 실습하고 있으며,
장기/방탈출/서바이벌 스타일의 웹·모바일 게임을 AWS 인프라와 연결하는 것을 목표로 하고 있습니다.

기술 스택

  • AWS: VPC, Subnet, IGW, NAT Gateway/Instance, EC2, ALB, ECS(EC2 & Fargate), ECR, S3, CloudFront, DynamoDB
  • Infra as Code: Terraform (VPC, EC2, SG, OpenVPN, ECS, ALB 등 모듈 구성 및 remote state)
  • CI/CD: GitHub Actions, GitLab CI, Docker 기반 GitLab Runner
  • 네트워크 & 보안: VPC Peering, NAT, OpenVPN, 보안 그룹, 라우팅 테이블 트러블슈팅
  • 개발: Python, C#, Unity(2D), HTML/CSS/JavaScript

프로젝트

AWS ECS + ALB Path 기반 라우팅 구축

2025 | 개인 학습 프로젝트

ECR에 저장된 여러 Docker 이미지를 ECS 서비스(EC2/Fargate 혼합)로 올리고,
ALB에서 /, /mzc/*, /aws/* 경로에 따라 서로 다른 서비스로 트래픽을 라우팅하는 아키텍처를 구성했습니다.

  • 프라이빗 서브넷의 ECS 인스턴스 + NAT GW 환경 구성
  • Auto Scaling Group으로 ECS EC2 용량 자동 조정
  • 타겟 그룹과 리스너 규칙을 이용한 Path 기반 라우팅
ECS ALB ECR Auto Scaling

Terraform 기반 VPC & 네트워크 모듈

2025 | 개인 인프라 랩

  • VPC, Subnet, Route Table, NAT, SG를 각각 모듈화

웹/모바일 게임 백엔드 AWS 연동 (진행 중)
