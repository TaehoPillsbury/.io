프로젝트 개요

고객사 신규 인프라 확장을 위해 Dell EMC PowerEdge R700 시리즈 서버를 납품하고, 초기 설치부터 디스크 RAID 구성, Hot Spare 설정, 하드웨어 교체(배터리, RAID Controller, NIC 등)까지 엔터프라이즈 서버 Life-cycle 전체를 직접 수행한 구축 사례입니다.

본 작업을 통해 고객사의 신규 서비스 운영 환경을 안정적으로 구성하고, 장애 발생 시 신속한 대응이 가능한 하드웨어 기반을 마련하였습니다.

  • 안정적 운영을 위한 지속적 유지보수 및 점검 체계 수립

구성 절차

1. 서버 납품 및 랙 설치

주요 내용

  • Dell EMC R710 / R720 / R730 등 R700 계열 모델 납품
  • 고객사 IDC 또는 본사 서버실에 장비 반입
  • 랙마운트 키트 설치(레일 장착)
  • 서버 DC/AC 기반 전원 PSU 연결
  • 듀얼 전원 구성(이중화) 적용

결과

  • 서버실 내 안정적인 하드웨어 설치 완료
  • N+1 전원 구조로 전원 장애 대비

2. 초기 설정(BIOS & iDRAC 구성)

작업 내용

  • BIOS 기본 설정 초기화 및 최적화
  • Boot mode (BIOS/UEFI)
  • Hyperthreading enable
  • Virtualization enable
  • Fan/thermal management 설정
  • iDRAC 초기 패스워드 설정
  • iDRAC Network 설정(IP, Gateway, DNS)
  • SNMP/Alert 설정

결과

  • 원격 관리(iDRAC) 기반의 운영/모니터링 가능
  • OS 설치 및 하드웨어 모니터링 준비 완료

3. 디스크 RAID 구성(PERC Controller)

주 RAID 작업

  • Dell PERC H700/H710/H730 RAID 컨트롤러 사용
  • 고객 서비스 목적에 따라 RAID 수준 선택
  • RAID 1 (OS용)
  • RAID 5 (데이터용, 비용 효율성)
  • RAID 10 (고성능 DB/트랜잭션 용도)

구성 절차

  • 서버 POST 단계에서 Ctrl + R 진입
  • Virtual Disk 생성
  • Disk Group 구성
  • Stripe Size, Read/Write 정책 설정
  • 초기화(Initialize) 실행

결과

  • 운영 목적에 최적화된 RAID 구성 적용
  • 디스크 장애 시 안정적 데이터 보호 확보

4. Hot Spare 디스크 설정

작업 내용

  • Global Hot Spare 또는 Dedicated Hot Spare 설정
  • RAID 5/6/10 등에서 장애 디스크 발생 시 자동 재빌드 가능하도록 구성

결과

  • 디스크 장애 발생 시 자동 복구
  • 운영 중단 없는 무중단 서비스 환경 지원

5. 하드웨어 유지보수(교체 및 점검)

수행한 유지보수 종류

① RAID Controller (PERC) 교체

  • RAID 컨트롤러 장애 감지(Log & iDRAC Alert)
  • 동일 모델로 교체 후 Virtual Disk 재인식 확인
  • Cache 정책/VD 상태 정상 복구

② Server Battery 교체 (PERC Battery)

  • 배터리 수명 경고 → 성능 저하 방지 위해 교체
  • Battery learn cycle 초기화 수행

③ NIC(네트워크 카드) 교체

  • 포트 Link Down / Packet Drop 문제 발생 시 교체
  • 1G/10G NIC 모두 유지보수 경험
  • PCIe 슬롯 재배치 및 드라이버 재설치

④ 팬/전원공급장치(PSU) 교체

  • Fan Failure 알람 발생 시 핫스왑 교체
  • PSU 장애 발생 시 듀얼 PSU 기반 교체 수행

⑤ 메모리/디스크 장애 처리

  • Faulty DIMM 교체 후 BIOS Memory Test 수행
  • 장애 난 HDD/SSD 교체 후 RAID rebuild 성공 여부 확인

6. 펌웨어 및 드라이버 업데이트

  • Lifecycle Controller 또는 Dell Repository Manager 사용
  • BIOS / iDRAC / PERC / NIC / PSU Firmware 업데이트
  • FW mismatch로 인한 장애 예방

결과

  • 최신 안정 버전 펌웨어 기반으로 장애 예방 및 성능 개선

7. OS 설치 및 기본 운영 환경 구축

  • VMware ESXi / Windows Server / Linux 설치
  • NIC Bonding / VLAN 구성
  • Hypervisor 초기 구성
  • 모니터링 에이전트 설치(Nagios, Zabbix 등)
