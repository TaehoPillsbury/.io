웹/모바일 게임 백엔드 AWS 연동 진행 중

진행 중

프로젝트 개요

방탈출/서바이벌 스타일의 웹 및 모바일 게임을 개발하고, 게임 내 점수 및 랭킹 데이터를 AWS 서비스를 활용하여 저장 및 관리하는 백엔드 시스템을 설계하고 있습니다.

서버리스 아키텍처를 기반으로 하여 초기 비용을 최소화하면서도 확장 가능한 구조를 목표로 하고 있습니다.

예상 아키텍처

프론트엔드

  • 정적 웹 호스팅: S3 + CloudFront를 통한 전역 CDN 구성
  • 게임 엔진: HTML/JavaScript 또는 Unity(2D) 활용
  • API 통신: RESTful API를 통한 백엔드와 통신

백엔드

  • API Gateway: RESTful API 엔드포인트 제공 및 인증
  • Lambda: 서버리스 함수를 통한 비즈니스 로직 처리
  • DynamoDB: 게임 점수 및 랭킹 데이터 저장

데이터 모델

  • 사용자 테이블: userId, nickname, createdAt 등
  • 점수 테이블: userId, score, timestamp, gameId 등
  • 랭킹 테이블: Global Ranking, Daily Ranking 등

구현 계획

1단계: 정적 웹 호스팅

  • S3 버킷 생성 및 정적 웹사이트 호스팅 설정
  • CloudFront 배포를 통한 HTTPS 및 CDN 구성
  • 도메인 연결 및 SSL 인증서 설정

2단계: DynamoDB 설계

  • 테이블 스키마 설계 (Partition Key, Sort Key)
  • GSI(Global Secondary Index)를 통한 랭킹 쿼리 최적화
  • TTL 설정을 통한 오래된 데이터 자동 삭제

3단계: Lambda 함수 개발

  • 점수 저장 함수 (PUT)
  • 랭킹 조회 함수 (GET)
  • 에러 핸들링 및 로깅

4단계: API Gateway 설정

  • REST API 생성 및 리소스 구성
  • Lambda 함수와 통합
  • CORS 설정

5단계: CI/CD 연동

  • GitHub Actions 또는 GitLab CI 파이프라인 구성
  • 코드 변경 시 자동 배포
  • 테스트 자동화

기대 효과

  • 서버리스 아키텍처를 통한 초기 비용 최소화
  • 자동 스케일링을 통한 트래픽 증가 대응
  • 전역 CDN을 통한 낮은 레이턴시
  • Infrastructure as Code를 통한 배포 자동화
  • 게임 백엔드 개발 및 운영 경험 축적

기술 스택

AWS API Gateway AWS Lambda DynamoDB S3 CloudFront JavaScript Unity CI/CD Game Backend
