VMware + LocalStack 기반 3-Tier 아키텍처 구축

2025 | 로컬 학습 환경

프로젝트 개요

비용 부담 없이 AWS 서비스를 학습하고 테스트하기 위해 VMware 17.5 Pro를 노트북에 설치하고, LocalStack을 활용하여 로컬 환경에서 AWS 서비스를 에뮬레이션하여 3-Tier 아키텍처를 구축했습니다.

LocalStack은 실제 AWS 서비스와 유사한 인터페이스를 제공하므로, 실제 AWS 환경으로 전환하기 전에 로컬에서 충분히 학습하고 테스트할 수 있는 환경을 제공합니다.

환경 구성

VMware 17.5 Pro

  • 노트북에 VMware 17.5 Pro 설치 및 라이센스 구성
  • Linux 가상머신 생성 및 네트워크 설정
  • 호스트와 가상머신 간 네트워크 통신 구성
  • 스냅샷 기능을 활용한 환경 백업 및 복원

LocalStack

  • Docker 컨테이너: LocalStack을 Docker로 실행
  • 지원 서비스: EC2, VPC, S3, DynamoDB, Lambda, API Gateway 등
  • 포트 매핑: 각 서비스별 포트 설정 및 접근 구성
  • 데이터 영속성: Docker 볼륨을 통한 데이터 저장

3-Tier 아키텍처 구성

Web Tier (프레젠테이션 계층)

  • 정적 웹 서버 또는 간단한 웹 애플리케이션
  • ALB(Application Load Balancer) 시뮬레이션
  • HTTPS 및 HTTP 트래픽 라우팅

Application Tier (애플리케이션 계층)

  • EC2 인스턴스 시뮬레이션 또는 컨테이너 기반 애플리케이션
  • 비즈니스 로직 처리 서버
  • API 서버 구성

Database Tier (데이터 계층)

  • DynamoDB 테이블 생성 및 데이터 저장
  • RDS 시뮬레이션 (LocalStack 지원 범위 내)
  • 데이터 백업 및 복구 테스트

구현 세부사항

네트워크 구성

  • LocalStack VPC 및 서브넷 생성
  • 보안 그룹 규칙 설정
  • 인터넷 게이트웨이 및 라우팅 테이블 구성

자동화

  • Terraform을 통한 LocalStack 인프라 코드 작성
  • AWS CLI를 LocalStack 엔드포인트로 설정하여 명령어 실행
  • 스크립트를 통한 환경 자동 구성

테스트 및 검증

  • 각 Tier 간 통신 테스트
  • 로드 밸런싱 동작 검증
  • 데이터 저장 및 조회 기능 테스트

장점 및 활용

  • 비용 절감: AWS 비용 없이 학습 가능
  • 빠른 반복: 로컬 환경에서 빠르게 테스트 및 수정 가능
  • 오프라인 학습: 인터넷 연결 없이도 학습 가능
  • 실제 환경과 유사: AWS CLI 및 Terraform과 호환
  • 안전한 실험: 실수해도 비용이나 실제 인프라에 영향 없음

학습 목표 및 성과

  • VMware를 활용한 가상화 환경 구성 경험
  • LocalStack을 통한 AWS 서비스 학습 및 테스트
  • 3-Tier 아키텍처의 실제 구현 및 이해
  • Terraform을 LocalStack과 함께 사용하는 방법 학습
  • 비용 부담 없이 충분한 학습 환경 구축
