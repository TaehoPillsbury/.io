AWS EC2(Free Tier) 기반 OpenVPN Server 구축

2025.11.25 | Marketplace AMI 없이, Amazon Linux 2023 기반 수동 설치

프로젝트 개요

AWS 프리티어 환경에서 비용을 최소화하면서, OpenVPN Access Server(유료 AMI)를 사용하지 않고 Amazon Linux 2023 + Easy-RSA + OpenVPN 조합으로 VPN 서버를 직접 구축한 프로젝트입니다.

Marketplace의 유료 OpenVPN AMI 대신 오픈소스 OpenVPN을 직접 설치하여 구성함으로써 비용 없이 VPN 서버를 구축하고, 실무적인 리눅스 서버 운영 및 네트워크 구성 경험을 쌓았습니다.

주요 구현 내용

1. Amazon Linux 2023 환경 구성

  • AWS EC2 Free Tier 인스턴스 생성 및 접속
  • Amazon Linux 2023 기본 패키지 업데이트
  • 필수 패키지 설치 (OpenVPN, Easy-RSA 등)

2. Easy-RSA 기반 PKI 구축

  • Easy-RSA 설치 및 PKI 디렉토리 초기화
  • CA(Certificate Authority) 인증서 생성
  • 서버 인증서 및 키 생성
  • 클라이언트 인증서 및 키 생성
  • Diffie-Hellman 키 교환 파일 생성

3. OpenVPN 서버 구성

  • OpenVPN 서버 설정 파일 구성 (server.conf)
  • TUN/TAP 인터페이스 구성
  • VPN 클라이언트 IP 대역 설정 (예: 10.8.0.0/24)
  • 암호화 및 인증 알고리즘 설정
  • OpenVPN 서비스 데몬화 및 자동 시작 설정

4. nftables 기반 NAT 설정

  • nftables 방화벽 규칙 구성
  • NAT(Network Address Translation) 설정을 통한 VPN 클라이언트 인터넷 트래픽 라우팅
  • IP 포워딩 활성화
  • 필요한 포트(1194/UDP 등) 오픈
  • 보안 그룹 규칙 설정

5. 클라이언트 자동화 스크립트

  • 클라이언트 인증서 자동 생성 스크립트 작성
  • OVPN 설정 파일 자동 생성 스크립트
  • 클라이언트별 고유 인증서 발급 프로세스 자동화
  • 생성된 .ovpn 파일 검증

6. S3를 통한 OVPN 파일 배포

  • AWS CLI를 통한 S3 버킷 생성 및 구성
  • 생성된 .ovpn 파일을 S3에 업로드
  • S3 버킷 정책 설정을 통한 안전한 다운로드 제공
  • 외부에서 OVPN 파일 다운로드 및 VPN 연결 테스트

구현 세부사항

네트워크 구성

  • EC2 인스턴스 퍼블릭 IP 및 Elastic IP 설정
  • 보안 그룹에서 UDP 1194 포트 인바운드 허용
  • VPC 라우팅 테이블 확인 및 설정

보안 구성

  • PKI 기반 인증서 인증으로 강력한 보안 구현
  • 암호화 알고리즘 최신 표준 적용
  • 서버 키 및 인증서 안전한 관리

자동화

  • Bash 스크립트를 통한 클라이언트 인증서 발급 자동화
  • AWS CLI를 통한 S3 업로드 자동화
  • 운영 효율성 향상을 위한 스크립트 최적화

프로젝트 결과 및 성과

  • 비용 0원 달성: OpenVPN 유료 AMI 없이 무료로 VPN 서버 구축
  • 최신 환경 적용: Amazon Linux 2023 + nftables 최신 기술 스택 활용
  • 인증서 기반 보안: Easy-RSA PKI 구성을 통한 강력한 보안 구현
  • 운영 자동화: 스크립트를 통한 클라이언트 관리 효율성 향상
  • VPN 접속 테스트 성공: 실제 VPN 연결 및 인터넷 트래픽 라우팅 검증
  • S3 기반 배포: 안전한 클라이언트 파일 배포 시스템 구축

학습 목표 및 경험

  • AWS EC2 프리티어 환경에서 비용 최적화 인프라 구성 경험
  • PKI(Public Key Infrastructure) 기반 인증 시스템 구축 및 이해
  • OpenVPN 서버 구성 및 운영 방법 학습
  • nftables를 통한 리눅스 방화벽 및 NAT 구성 경험
  • VPN 네트워크 아키텍처 이해 및 트래픽 라우팅 원리 학습
  • AWS CLI 및 S3를 활용한 파일 배포 자동화 경험
  • 리눅스 서버 관리 및 네트워크 구성 실무 경험 축적
